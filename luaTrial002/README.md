# uecsOS on Teensy 4.1 (Lua Integration Edition)

Teensy 4.1 をベースとした、スマート農業向け環境制御システムのプロトタイプです．
最大の特徴は、C++による堅牢なシステム基盤の上に **Lua 5.5 インタプリタ** を統合している点にあります．これにより、ハードウェアの再書き込みなしに、ネットワーク経由で動的に制御ロジックを変更することが可能です．

## 🚀 主な機能

* **Lua 5.5 フル統合**: 軽量・高速なスクリプト言語 Lua を利用してハードウェアを制御．
* **オートラン機能 (`startup.lua`)**: 起動時に SD カードから `startup.lua` を自動読み込み・実行．
* **UDP ライブ・コーディング**: ネットワーク経由（ポート 8888）で Lua コードを送信し、即座に実行．
* **ネイティブ・ハードウェア操作**: Lua から以下の機能を直接呼び出せます．
* **GPIO**: `digitalWrite` による LED やリレーの操作．
* **I2C**: `lcd_init`, `lcd_print` 等による 20x4 LCD 制御、および汎用 I2C 通信．
* **SD Storage**: `sd_read`, `sd_append`, `dir` によるデータログ取得とファイル管理．


* **DHCP/UDP**: `NativeEthernet` を利用した自動 IP 取得と通信．

## 🛠 ハードウェア構成

* **MCU**: Teensy 4.1
* **Display**: I2C 接続 20x4 LCD (Address: `0x3F`)
* **Storage**: 内蔵 MicroSD スロット
* **Network**: Teensy 4.1 内蔵 Ethernet (DP83825 PHY)

## 📂 ファイル構成

* `main.cpp`: システムのメインロジック（Ethernet/UDP 管理、Lua インスタンスの生成・実行）．
* `lua_functions.cpp / .h`: C++ で定義したハードウェア制御関数を Lua に登録するブリッジ層．
* `src/lua/`: Lua 5.5 ソースコード本体．

## 📝 使い方

### 1. 起動時スクリプトの配置

SD カードのルートディレクトリに `startup.lua` を配置します．このスクリプト内で、起動時の LCD 表示や初期設定を記述できます．
`main.cpp` は起動時に `my_ip` というグローバル変数に取得した IP アドレスをセットするため、スクリプト内で参照可能です．

### 2. リモートからのコード実行

ネットワーク上の PC から UDP でコードを送信します．最後に `.` (ドット) だけの行を送ると、蓄積されたバッファが実行されます．

```bash
# 例: LCDをクリアしてメッセージを表示
echo -n 'lcd_clear(); lcd_print("Remote Update");' | nc -u -w1 <TEENSY_IP> 8888
echo -n "." | nc -u -w1 <TEENSY_IP> 8888

```

## ⚠️ 開発上の注意

* **IP Address 変換**: Teensy の `IPAddress` クラスには `toString()` がないため、各オクテットを個別に文字列結合して Lua へ渡しています．
* **リセット機能**: `reset()` 関数を呼び出すことで、ARM Cortex-M7 のシステムリセットを実行し、システム全体を再起動（`startup.lua` から再開）できます．

## 📚 Lua 関数リファレンス

システムに登録されており，`startup.lua` や UDP 経由で直接呼び出し可能な関数の一覧です．

### 📟 液晶ディスプレイ (LCD 20x4)

I2C アドレス `0x3F` のディスプレイを制御します．

* **`lcd_init()`**: LCD を初期化し，バックライトを点灯させます．
* **`lcd_clear()`**: 画面表示をすべて消去します．
* **`lcd_print(text)`**: 現在のカーソル位置から文字列を表示します．
* **`lcd_setCursor(col, row)`**: 指定した位置（列: 0-19, 行: 0-3）にカーソルを移動します．

### 💾 SD カードストレージ

内蔵 MicroSD スロットのファイルを操作します．

* **`dir([path])`**: 指定したディレクトリ内のファイル一覧をテーブルで返します（デフォルトは `/`）．
* **`sd_read(filename)`**: ファイルの内容をすべて読み込み，文字列として返します．失敗時は `nil` を返します．
* **`sd_append(filename, text)`**: 指定したファイルに文字列を追記します．成功時に `true` を返します．

### 🔌 ハードウェア制御

Teensy 4.1 の基本機能を操作します．

* **`digitalWrite(pin, state)`**: 指定したピンの出力を `1` (HIGH) または `0` (LOW) に設定します．
* **`delay(ms)`**: 指定したミリ秒の間，処理を停止します．
* **`print(...)`**: シリアルモニタ（USB）に文字列を出力します．複数の引数はタブ区切りで表示されます．
* **`reset()`**: Teensy 本体をソフトウェアリセットし，再起動させます．

### 📡 I2C 低レイヤ通信

汎用的な I2C デバイスの制御に使用します．

* **`i2c_begin()`**: I2C 通信（Wire）を初期化します．
* **`i2c_write(addr, reg, val)`**: 指定アドレスのレジクタに値を書き込みます．戻り値 `0` は成功を意味します．
* **`i2c_read(addr, len)`**: 指定アドレスから指定バイト数分データを読み込み，テーブル形式で返します．

### 🌐 システム変数

* **`my_ip`**: (文字列) DHCP で取得した現在の IP アドレスが自動的に格納されています．
